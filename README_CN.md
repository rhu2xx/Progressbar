# ProgressBar - 桌面时间进度条

一个简洁美观的桌面时间进度条工具，实时显示当前时间和时间段完成进度。

## 功能特性

- ✨ **实时进度显示** - 根据设定的时间段显示当前进度百分比
- 🎨 **自定义外观** - 可调整窗口大小、透明度、颜色
- 📍 **边缘吸附** - 自动吸附到屏幕四边
- 🔝 **窗口置顶** - 可选择是否始终显示在最前
- ⏰ **时间段设置** - 自定义开始和结束时间
- 📊 **刻度显示** - 可选的时间刻度线
- 🚀 **开机自启** - 支持设置开机自动启动
- 💾 **便携式** - 单文件 exe，无需安装

## 使用方法

### 直接运行

双击 `ProgressBar.exe` 即可运行（位于 `dist` 文件夹）。

### 从源码运行

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行程序：
```bash
python main.py
```

## 功能说明

### 右键菜单

在进度条上右键点击可打开菜单：

| 菜单项 | 说明 |
|--------|------|
| **设置时间段** | 设置开始和结束时间（默认 10:00-22:00） |
| **设置大小** | 调整进度条的宽度（200-2000px）和高度（20-200px） |
| **设置透明度** | 调整窗口透明度（10-100%） |
| **✓ 吸附开关** | 开启/关闭屏幕边缘自动吸附 |
| **✓ 置顶开关** | 开启/关闭窗口始终置顶 |
| **✓ 显示刻度** | 显示/隐藏时间刻度线（12段） |
| **✓ 开机自启** | 设置/取消开机自动启动 |

> ✓ 表示该选项当前已启用

### 拖动窗口

- 按住左键拖动可移动窗口位置
- 松开鼠标时会自动吸附到屏幕边缘（如果启用了吸附功能）

### 显示内容

进度条中央显示：
- 当前时间（24小时制）
- 完成百分比

例如：`15:30 65%`

## 打包说明

使用 PyInstaller 打包为单文件 exe：

```bash
python -m PyInstaller --onefile --windowed --name "ProgressBar" main.py
```

生成的 exe 文件位于 `dist` 文件夹。
